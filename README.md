# SAWINFOTECH Legacy Website

## Overview
This repository contains the source for the legacy SAWINFOTECH personal website (https://legacy.sawinfotech.com), built and hosted as an ASP.NET Web Forms site. The home page introduces the site as Sunit Waingankar's personal portal and a collection of web applications, along with personal/community content under SAWINFO.

## Technical Stack
- Platform: ASP.NET Web Forms (.aspx) with code-behind in C#
- Framework: .NET Framework 4.7.2
- Data access: Entity Framework 6.4.4
- Database providers/storage:
  - SQL Server
  - Microsoft Access files 
  - SQL CE file 
- Authentication: ASP.NET Forms Authentication + Membership/Role providers
- Web services: ASMX endpoint with AJAX usage
- Front-end: HTML/CSS, JavaScript, jQuery 1.4.1
